# My Java Prep

A structured repository dedicated to mastering Core Java, Object-Oriented Programming (OOPs), and standard layout/graphics implementations. This repository tracks my academic and practical journey through Java fundamentals, data structures, and terminal-based compilation workflows.

---

## 📂 Repository Structure

*   `core-java/` — Basics, syntax, loops, and conditional statements.
*   `oops/` — Concepts like Inheritance, Polymorphism, Encapsulation, and Abstract classes.
*   `constructors/` — Deep dive into Default, Parameterized, and Copy constructors.
*   `awt-swing/` — GUI layouts, components, and event handling.
*   `algorithms/` — Core algorithmic implementations and complexity analysis.

---

## 🛠️ Getting Started

### Prerequisites
Make sure you have the Java Development Kit (JDK) installed on your machine. You can verify this in your terminal:


check java version 

javac -version

compile the code

javac filename.java

for running the codes 
java filename.java
